Diffie-Hellman key exchange is commonly used in many systems that need to create a secure communication channel. One example is the TLS handshake
which happens when you connect to a secure website. It helps the browser and the server agree on a shared secret key so that the data they send
back and forth stays private.
Modern versions like ECDHE make this process faster and more secure. Another good example is secure messaging apps like Signal and WhatsApp which
use similar key exchanges to keep conversations encrypted and private.
